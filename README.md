# ¿Qué es OpenDaylight (ODL)? 
# El proyecto OpenDaylight es una plataforma de código abierto para redes definidas por software (SDN) que utiliza 
protocolos abiertos para proporcionar control programático centralizado y 
monitoreo de dispositivos de red.
